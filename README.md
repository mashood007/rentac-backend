## Car Rental API
## Description
RentAC is offering various mobility solutions to several customer segments from short term renting for tourists and UAE residents to long term leasing for large corporate and individual users.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Stack
- Nest Js
- PostgreSQL
- Prisma
- TypeScript
